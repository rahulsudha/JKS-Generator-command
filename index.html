Help - JKS/CSR Command Generator
About

JKS/CSR Command Generator helps to generate UNIX CLI commands to create java keystore files and certificate signing requests

Usage

For best result use Chrome browser.
Make sure no unexpected spaces are there in the details provided in the input fields.
Once all the data has been entered, press Parse button to process the command.
Use appropriate copy buttons to copy the JKS/CSR.
Use the Clear all button near the name field to clear all entries

Once the command has been copied, Simply paste the same in the required directory in the unix box.

Note: The result contains an empty line, so when pasted in the CLI it automatically makes an "enter" button action.
